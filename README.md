## Project Overview

This is a professional portfolio website showcasing the works of **John Carter**. The site includes multiple pages, such as Home, About, Books, Blog, and Contact. It offers a seamless user experience with responsive design and various interactive features.

## Features

- Home, About, Books, Blog, and Contact pages
- Responsive layout (mobile, tablet, desktop)
- Dark/Light theme toggle
- Search functionality for books and blog posts
- Contact form with EmailJS integration and validation
- Smooth animations with Framer Motion
- Local image assets management

## Tech Stack

- `React`
- `Vite`
- `react-router-dom`
- `tailwindcss`
- `framer-motion`
- `react-hook-form`
- `@emailjs/browser`
- `react-toastify`
- `react-icons`

## Project Structure

- `src/pages`: Route pages (`Home`, `About`, `Books`, `Blog`, `Contact`)
- `src/components`: Shared UI components (`Navbar`, `Footer`, `Layout`)
- `src/context`: Theme context provider
- `src/data/mockData.js`: Content objects (author details, books, blog posts)
- `src/assets`: Local image assets
